## ✨ Overview
**Career App** is a **full-stack MERN application** that helps users discover career paths based on their strengths and preferences. Take a quiz, get personalized career suggestions, and plan your roadmap!  

**Key Highlights:**  
- 🔒 Secure **User Authentication** with JWT  
- 📝 Interactive **Quiz Flow**  
- 🎯 Personalized **Career Results** (Skills, Courses, Roadmap)  
- 💻 Responsive **React UI** for desktop and mobile  

---

## 🛠 Features
- **User Authentication** – Register & login securely  
- **Profile Management** – Save session in `localStorage`  
- **Career Quiz** – 10 engaging career-related questions  
- **Result Generation** – Personalized recommendations:  
  - Match Percentage  
  - Recommended Skills  
  - Suggested Courses  
  - 5-Year Roadmap  
- **Navigation** – Smooth previous/next buttons with progress tracking  
- **Responsive Design** – Optimized for all devices
